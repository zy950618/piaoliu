# 企业级生产验收门禁 v2

日期：2026-06-29

## 总规则

**没有证据的通过，一律按未通过处理。**

证据包括但不限于：

- 测试命令输出。
- 接口请求和响应。
- 截图或 Playwright 断言。
- 数据库查询结果。
- 审计日志记录。
- 文档更新位置。
- 错误和失败项说明。
