# Vue3 Admin UX Review

Check admin-web only. Never move admin pages into user src pages.

## Review checklist

- Role-based menu.
- Table density.
- Advanced filters.
- Batch action bar.
- Detail drawer.
- Audit trail.
- Error state with request_id.
- Loading/empty states.
- Keyboard and focus behavior.
