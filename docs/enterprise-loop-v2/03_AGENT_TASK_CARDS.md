# Agent 任务卡总表 v2

每张任务卡都可以单独复制给一个子 Agent。所有 Agent 必须遵守总提示词、LOOP 标准和验收门禁。


## A0-Orchestrator：总控 Agent

### 负责范围

读取项目、冻结规则、拆分任务、控制依赖、合并验收

### 必须先读

```text
README.md
docs/**
与本 Agent 相关的代码目录
上一轮 LOOP 报告
```

### 禁止事项

不直接大范围写业务代码；只在冲突和整合时改最小范围

### 必须输出

LOOP 计划、依赖图、风险清单、最终验收报告

### 验收证据

```text
读取文件清单：
修改文件清单：
测试命令：
接口/页面证据：
失败项：
需要接力：
是否可合并：YES/NO
```

### 通过标准

- 没有违反不可变业务规则。
- 有自动化或手工证据。
- 文档已同步。
- 影响其他 Agent 的接口/字段/状态已通知。
