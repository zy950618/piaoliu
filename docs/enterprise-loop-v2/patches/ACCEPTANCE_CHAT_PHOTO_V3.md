# 上下文私聊与私密照片智能审核验收清单 V3

## A. 上下文私聊

### 必须通过

- [ ] 无上下文时，陌生人不能直接私聊。
- [ ] 漂流瓶回应后，瓶子作者回复/确认继续聊后可以开启会话。
- [ ] 广场/树洞评论后，作者回复/确认继续聊后可以开启会话。
- [ ] 游戏、房间、扩列上下文中，双方确认或完成互动后可以开启临时会话。
- [ ] 好友关系不再是私聊唯一门槛。
- [ ] 非好友上下文会话必须有来源卡片。
- [ ] 会话记录必须保存 `source_type`、`source_id`、参与人、创建时间和解锁原因。
- [ ] 被拉黑、封禁、风控命中用户不能开启或继续会话。
- [ ] 非好友上下文会话有频率限制。
- [ ] 聊天命中违规词时可以拦截、限流或进入审核。
- [ ] 后台能看到会话来源和举报证据。
- [ ] 前端能解释“为什么可以聊天”和“如何退出/拉黑/举报”。

### 不能通过

- [ ] 仍要求所有私聊必须先加好友。
- [ ] 允许任意用户从用户列表直接私聊陌生人。
- [ ] 没有来源追踪。
- [ ] 没有拉黑和举报。
- [ ] 没有频控。
- [ ] 没有测试覆盖。

## B. 私密照片智能审核

### 必须通过

- [ ] AI 审核优先。
- [ ] 低风险高置信内容自动通过。
- [ ] 中风险内容进入人工复核。
- [ ] 高风险内容拒绝或冻结。
- [ ] 用户举报触发二次审核。
- [ ] 收益异常触发二次审核。
- [ ] 所有审核结果保留模型标签、置信度、风险等级、动作原因和时间。
- [ ] 人工复核可以覆盖 AI 结果。
- [ ] 被拒绝或冻结内容不能展示。
- [ ] 被拒绝或冻结内容不能产生收益。
- [ ] 后台可按风险等级、状态、用户、时间、举报数筛选。
- [ ] 前端展示审核中、已通过、未通过、申诉中等状态。
- [ ] 重复上传不能绕过审核。
- [ ] 审计日志完整记录状态变化。

### 不能通过

- [ ] 所有内容都人工先审。
- [ ] 所有内容都无审核展示。
- [ ] 没有风险分级。
- [ ] 没有模型证据。
- [ ] 没有人工复核兜底。
- [ ] 没有举报二次审核。
- [ ] 没有收益冻结规则。
