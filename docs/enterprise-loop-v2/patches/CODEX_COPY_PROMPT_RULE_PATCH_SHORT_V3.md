需要替换旧规则：

1. 旧规则“陌生人不能直接私聊 / 好友申请通过后才开放聊天”不准确。
2. 正确规则是：禁止无上下文冷启动骚扰，但允许明确互动上下文内的陌生人私聊。
3. 漂流瓶场景：B 捞到 A 的瓶子并回应后，A 回复该回应或点击“继续聊/私聊”，双方可以开启上下文私聊，无需先加好友。
4. 广场/树洞场景：B 评论 A 的内容后，A 回复该评论或点击“继续聊/私聊”，双方可以开启上下文私聊，无需先加好友。
5. 游戏/房间/扩列场景：双方进入同一房间或匹配上下文，并完成明确互动或确认后，可以开启临时私聊。
6. 好友关系不是私聊唯一入口。好友关系用于长期关系、更多资料可见性、更低频控限制和后续关系沉淀。
7. 仍然禁止无上下文直接骚扰陌生人，必须有来源、双向回应/确认、频控、拉黑、举报、风控和审计。

私密照片规则也要改：

1. 旧规则“私密照片必须先人工审核后展示”不合理。
2. 正确规则是：AI 智能审核优先，风险分级，人工复核兜底。
3. 低风险、高置信、非露骨、无敏感隐私、无未成年人疑似、无诈骗导流内容，可以自动通过。
4. 中风险、低置信、边界内容进入人工复核。
5. 高风险内容自动拒绝或冻结。
6. 用户举报、收益异常、风控异常、重复违规会触发二次审核。
7. 所有审核必须保留模型标签、置信度、风险等级、自动动作、人工复核记录和审计日志。
8. 收益只允许来自审核通过且未冻结的内容。
